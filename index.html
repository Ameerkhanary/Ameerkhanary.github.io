Ask Me Anything.

What inspired you to combine AI with Aerospace?
I realized that while aerodynamics provides the body, AI provides the brain. My goal is to create autonomous systems that can "see" and "think" like pilots, using tools like Computer Vision to navigate without GPS.
How do you optimize AI for Jetson Nano?
I use TensorRT optimization and quantization (FP16/INT8) to reduce model size without losing significant accuracy. This allows YOLOv8 to run in real-time (30+ FPS) on edge devices.
What is your Patent about?
"An Autonomous Guidance and Tracking System with a Distributed AI-Edge Computing Architecture." Essentially, it shifts the processing load from ground stations to the missile/drone itself, enabling true autonomy.
